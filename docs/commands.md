## Instalation

### Usage

For download a video:
```
youd <youtube URL> 
```

or

For download a specific format:
```
youd <youtube URL> --mp3
```

```
youd <youtube URL> --mp4
```

### Download location

#### Windows
`Users/{User}/Documents/Youd`

#### MacOS
`Users/{User}/Documents/Youd`

#### Linux
`home/{User}/Youd`
